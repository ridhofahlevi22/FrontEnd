## Instalation 
*Pastikan sudah terinstall nodejs

1. Clone repository
```shell
git clone https://github.com/ridhofahlevi22/FrontEnd.git
```
2. Get in the project folder
```shell
cd FrontEnd
```
3. Install dependencies via npm or yarn
```shell
npm install
```
```shell
yarn
```

## Quick start
Run development server
```shell
yarn run
```
